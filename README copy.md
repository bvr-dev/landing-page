## 🚀 Características
- **Multilingüe**: Integración completa UI con soporte bilingüe (Español / Inglés) vía React Context usando estado global.
- **Micro-animaciones**: Diseñado exclusivamente con CSS Vainilla (`@keyframes`, glassmorphism y variables dinámicas).
- **Responsivo**: Adaptable a dispositivos móviles y escritorios para una excelente experiencia de usuario.
- **Rápido**: Configurado y empaquetado bajo el poder de Vite.js.

## 🛠 Instalación y Uso Local
1. Clona este repositorio:
   ```bash
   git clone https://github.com/Bertin2/Landing-page.git
   ```
2. Entra al directorio:
   ```bash
   cd Landing-page
   ```
3. Instala las dependencias necesarias:
   ```bash
   npm install
   ```
4. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```

## 🌐 Despliegue
Este proyecto fue construido para ser desplegado fácilmente en plataformas de alojamiento de sitios estáticos gratuitos como **Render**, **Vercel** o **Netlify**. Asegúrate de apuntar tu comando de construcción hacia `npm run build` y publicar la carpeta de distribución (`/dist`).
